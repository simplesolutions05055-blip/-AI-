# תיעוד: כפתור הצגת/הסתרת סיסמה

**תאריך:** 2026-07-01 08:32
**מבצע:** Claude Code

## מה נדרש
להוסיף במסך ההתחברות ובמסך ההרשמה כפתור (אייקון עין) שאפשר ללחוץ עליו כדי לראות את הסיסמה שהוקלדה.

## מה נעשה
נוסף כפתור toggle של הצגת/הסתרת סיסמה בשני המסכים:

- **מסך כניסה** — `src/pages/LoginPage.tsx`
- **מסך הרשמה** — `src/pages/SignupPage.tsx`

### פרטי המימוש
- הכפתור משתמש באייקונים `Eye` / `EyeOff` מ-`lucide-react` (חבילה שכבר קיימת בפרויקט).
- נוסף state חדש `showPassword` שמחליף את `type` של שדה הסיסמה בין `password` ל-`text`.
- שדה הסיסמה עוטף ב-`div` עם `position: relative`, והכפתור ממוקם באופן אבסולוטי.
- **מיקום (RTL):** הכפתור בצד **שמאל** של השדה (`end-0`), בעוד הסיסמה נשארת מיושרת לימין. הריפוד `pe-10` משאיר מקום לאייקון בצד שמאל.
- **נגישות:** תגית `aria-label` מתחלפת דינמית — "הצג סיסמה" כשמוסתר, "הסתר סיסמה" כשגלוי. הכפתור מסוג `type="button"` כדי לא לשלוח את הטופס בטעות.

### קטע קוד מייצג
```tsx
const [showPassword, setShowPassword] = useState(false);

<div className="relative mb-4">
  <input
    id="password"
    type={showPassword ? 'text' : 'password'}
    ...
    className="w-full rounded-lg border border-[var(--border)] pe-10 ps-3 py-2"
  />
  <button
    type="button"
    onClick={() => setShowPassword((v) => !v)}
    aria-label={showPassword ? 'הסתר סיסמה' : 'הצג סיסמה'}
    className="absolute inset-y-0 end-0 flex items-center px-3 text-[var(--muted)] hover:text-[var(--foreground)]"
  >
    {showPassword ? <EyeOff className="h-5 w-5" /> : <Eye className="h-5 w-5" />}
  </button>
</div>
```
