# מדריך הצ׳אט — מה אפשר לבקש ואיך זה עובד

מסמך זה מסביר בשפה פשוטה מה הסוכן (הצ׳אט) יודע לעשות, איך מתנהלת השיחה, מה מקבלים בכל אופציה, ואיזה קבצים ותוכן אפשר להזין.

---

## איך זה עובד — בגדול

הצ׳אט מתנהג כמו **סוכן AI בוואטסאפ**. כותבים לו בשפה חופשית מה רוצים, והוא מוביל את התהליך:

1. **שואל שאלות הבהרה** — אם חסר לו מידע כדי להבין מה צריך.
2. **בונה בריף** — סיכום מסודר של הבקשה (מטרה, קהל יעד, מסרים וכו').
3. **מפיק את התוצר** — כשהבריף מוכן, מופיע כפתור "אשר והפק", ולחיצה עליו מייצרת את התוצר בפועל.

בנוסף, הסוכן **מזהה אוטומטית שם של עיר/מותג** בתוך מה שנכתב, ושולף ממסד הנתונים את פלטת הצבעים, הלוגו וסגנון העיצוב שלו. כל התוצרים יוצאים ממותגים בהתאם.

---

## מה אפשר לבקש — 3 סוגי תוצרים

### 1. תמונה

* **מתי משתמשים:** בקשה לפוסט, באנר, תמונה לאירוע, קמפיין או הודעה עירונית.
* **איך מתנהל התהליך:**
  1. הצ׳אט שואל שאלות הבהרה ובונה בריף עיצובי.
  2. מופיע כפתור **"אשר והפק תמונה מהבריף"**.
  3. נוצרת תמונה לפי הבריף.
  4. אם קיים לוגו של העיר במערכת — **הלוגו האמיתי מודבק על התמונה** (כדי להבטיח לוגו מדויק).
* **מה מקבלים:** תמונת PNG להורדה, ממותגת בצבעי העיר.

### 2. מצגת — מצב NotebookLM / Gemini

* **מתי משתמשים:** בקשה למצגת, כשרוצים להמשיך לבנות אותה בכלי חיצוני.
* **איך מתנהל התהליך:**
  1. הצ׳אט שואל **בהודעה אחת** את כל השאלות החיוניות (מטרה, קהל יעד, מסרים מרכזיים, קריאה לפעולה).
  2. מופיע כפתור **"הפק פרומפט ל-Gemini / NotebookLM"**.
  3. מתקבל תוכן מובנה: כותרות, תוכן לכל שקף, הנחיות עיצוב, ופרומפט מוכן להדבקה.
  4. אם לעיר יש תמונות מיתוג ולוגו — מצורפים **קישורי הורדה פתוחים, תקפים ל-7 ימים**.
* **מה מקבלים:** טקסט שמעתיקים ל-NotebookLM / Gemini, והם בונים את המצגת.

### 3. מצגת — מצב Deck ממותג (PDF)

* **מתי משתמשים:** כשרוצים מצגת מוכנה כקובץ, בלי כלי חיצוני.
* **איך מתנהל התהליך:**
  1. אותו תהליך שאלות ובריף.
  2. מופיע כפתור **"הכן והורד מצגת ממותגת"**.
  3. נוצרת מצגת של **בדיוק 10 שקפים** (כותרת, נקודות, גוף, הצעת תמונה לכל שקף).
  4. נבנה **קובץ PDF אמיתי** בעברית (RTL), בצבעי העיר, עם תמונות המיתוג מוטמעות.
* **מה מקבלים:** קובץ PDF שיורד אוטומטית, וגם פרומפט ל-NotebookLM לשחזור המצגת בכלי חיצוני.

## מה אפשר להזין לצ׳אט

הצ׳אט יודע לקבל קבצים ולהבין את תוכנם. כל קובץ מומר אוטומטית לטקסט שהסוכן קורא ומתייחס אליו:

| סוג קלט | מה קורה איתו |
|---|---|
| **טקסט חופשי** | תיאור הבקשה: מטרה, קהל יעד, מסרים, קריאה לפעולה, אימייל ליצירת קשר. |
| **תמונה** (jpg / png) | מנותחת על ידי מודל Vision — הסוכן מקבל תיאור מפורט של מה שמופיע בתמונה (כולל טקסט שבתוכה). |
| **מסמך** (docx / txt / md) | התוכן נקרא במלואו ומוזן לסוכן כדי שיתבסס עליו. |
| **אודיו** | מתומלל אוטומטית לטקסט. לאחר התמלול מופיע כפתור **"הצג תמלול"** בהודעה, לצפייה במה שתומלל. |
| **שם עיר / מותג** | אזכור של מותג שקיים במערכת מושך אוטומטית את הצבעים, הלוגו והסגנון שלו. |

> **מגבלת גודל:** ניתן להעלות קבצים עד **25MB**. קובץ גדול יותר ייחסם, ותוצג הודעה מתאימה.

### בחירת מודלים (מסך הניהול)
* **מודל תמלול אודיו** — `gpt-4o-transcribe` (ברירת מחדל) / `gpt-4o-mini-transcribe` / `whisper-1`.
* **מודל הבנת תמונות (Vision)** — `gpt-4o` / `gpt-4o-mini`.
