# המשך פיתוח: תזמון פרסום לפייסבוק ולאינסטגרם

## מצב נוכחי

נוסף אזור "תזמון פרסום" בעמודי התוצרים, עם כפתורים לפייסבוק ולאינסטגרם.
כל כפתור פותח modal מיושר RTL עם שדות תאריך/שעה וכיתוב לפרסום.
כפתור התזמון עצמו חסום כרגע, והמשתמש רואה הודעה שהתזמון עדיין לא זמין בשלב הזה.

## מה צריך להוסיף בהמשך

1. הגדרת אינטגרציה מול Meta
   - פתיחת אפליקציה ב-Meta for Developers.
   - חיבור Facebook Login / Instagram Graph API.
   - הגדרת הרשאות מתאימות לפרסום ותזמון.
   - הגדרת redirect URLs לסביבת פיתוח ולסביבת production.

2. חיבור חשבונות משתמשים
   - מסך או פעולה לחיבור עמוד פייסבוק וחשבון אינסטגרם עסקי.
   - שמירת מזהי העמודים והחשבונות המחוברים.
   - שמירת access tokens בצורה מאובטחת בצד השרת בלבד.
   - טיפול ברענון או החלפת tokens לפי דרישות Meta.

3. מודל נתונים לתזמונים
   - טבלת `scheduled_social_posts`.
   - שדות מוצעים:
     - `id`
     - `request_id`
     - `output_id`
     - `platform`
     - `caption`
     - `scheduled_at`
     - `status`
     - `external_post_id`
     - `error_message`
     - `created_by`
     - `created_at`
     - `updated_at`

4. Edge Function ליצירת תזמון
   - קבלת `request_id`, פלטפורמה, תאריך ושעה, וכיתוב.
   - בדיקת הרשאות המשתמש לתוצר.
   - בדיקת תקינות התאריך, כולל מניעת תזמון לעבר.
   - שמירת רשומת תזמון בסטטוס `scheduled`.
   - החזרת הודעת הצלחה לממשק.

5. Worker לפרסום בפועל
   - תהליך מתוזמן שמאתר פוסטים שהגיע זמנם.
   - הורדת התוצר או יצירת URL חתום לקובץ.
   - פרסום דרך Graph API.
   - עדכון סטטוס ל-`published` או `failed`.
   - שמירת שגיאות מלאות ללוג פנימי והצגת הודעה קצרה למשתמש.

6. התאמות UI
   - הפיכת כפתור "תזמון הפרסום" לפעיל אחרי שיש אינטגרציה.
   - הצגת מצב תזמון: מתוזמן, פורסם, נכשל.
   - אפשרות ביטול תזמון לפני מועד הפרסום.
   - אפשרות עריכת תאריך/שעה וכיתוב כל עוד הפרסום לא בוצע.

7. בדיקות
   - בדיקת תזמון לפייסבוק עם תמונה, טקסט ומצגת/קובץ.
   - בדיקת תזמון לאינסטגרם עם תמונה וכיתוב.
   - בדיקה שתאריך עבר נחסם.
   - בדיקה שמשתמש לא יכול לתזמן תוצר שאינו שייך לו.
   - בדיקה שכשפרסום נכשל, הסטטוס והודעת השגיאה נשמרים.

## החלטות מוצר שצריך לקבל

- האם תזמון באינסטגרם יתמוך רק בתמונה, או גם בוידאו בעתיד.
- האם פוסט טקסטואלי בלבד יפורסם לפייסבוק בלבד.
- האם כל משתמש מחבר חשבונות משלו, או שהמערכת מפרסמת דרך חשבונות מרכזיים.
- האם צריך אישור נוסף לפני פרסום אוטומטי.
- האם לאפשר תזמון חוזר או רק פרסום חד-פעמי.
